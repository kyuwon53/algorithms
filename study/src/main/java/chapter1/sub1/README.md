## 1.1.1 표현식의 결괏값은 무엇인가?
### a. ( 0 + 15) / 2 
#### 답. 7   
해설: 
연산의 결과가 그 연산의 대상이 되는 데이터 타입과 동일한 데이터 타입을 가진다 (닫혀있다)

### b. 2.0e-6 * 100000000.1 
#### 답. 200.0000002
해설:
숫자는 그 숫자의 정밀도를 잃어버리지 않는 범위 안에서 가장 작은 표현 범위의 데이터 타입으로 자동 변환된다. 

### c. true && false || true && true 
#### 답. ture
